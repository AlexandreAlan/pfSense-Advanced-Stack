# pfSense-Advanced-Stack
Framework avançado de infraestrutura para pfSense 2.7.2+. Implementação de Traffic Shaping (FQ_CODEL), Captive Portal escalável (FreeRADIUS/MySQL), WPAD/Proxy transparente, hardening de firewall HTTPS, gestão de identidades e automação de Blacklists granulares para conformidade com a LGPD e alta disponibilidade.
